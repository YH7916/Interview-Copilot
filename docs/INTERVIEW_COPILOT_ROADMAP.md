# Interview Copilot Roadmap

## Current State

The core harness loop is now in place:

- prep pack generation
- project-centric live mock interview
- structured review
- next-drill generation
- prep and interview traces

## Next Technical Steps

### 1. Replay And Benchmark

Add stronger offline comparison for:

- prompt variants
- retrieval variants
- follow-up policy variants

### 2. Better Drill Scheduling

Turn review results into:

- recurring drill queues
- targeted mock sessions
- spaced repetition for weak areas

### 3. Stronger Target Modeling

Improve target-role reasoning with:

- richer JD parsing
- company-specific loop templates
- more explicit role-to-project mapping

### 4. Better Demo Surface

Keep the product simple, but make it easier to show:

- one end-to-end prep example
- one trace example
- one mock-review-drill example

## Product Principle

Stay focused on:

- interview preparation
- runtime quality
- retrieval grounding
- traces and evaluation

Do not expand into:

- a giant generic multi-agent platform
- a heavy frontend-first product
- many unrelated verticals
